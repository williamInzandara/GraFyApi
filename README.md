# 🧮 graFyApi — Visualizador Matemático 3D

## ✨ Funcionalidades

- 🌐 **Visualización 3D interactiva** de superficies f(x, y, t) con Three.js
- 📐 **Modos de visualización**: Interactivo, Campo de Gradiente, Análisis, Intersección
- 🎛 **8 superficies predefinidas**: Onda Ondulante, Paraboloide Hiperbólico, Sombrero Mexicano, etc.
- 🧠 **Calculadoras matemáticas**: Derivadas parciales, Límites, Optimización Lagrange, Integración doble
- 📊 **Visualización 2D de regiones** con curvas de nivel y restricciones
- ⚙️ Control de rango de dominio y resolución de malla en tiempo real
- 🌀 Soporte para parámetro de tiempo `t` con animación continua

## 🛠 Stack Tecnológico

| Tecnología | Uso |
|:-----------|:----|
| **Next.js 15** | Framework React con App Router |
| **TypeScript** | Tipado estático |
| **React 18** | UI y estado |
| **Three.js** | Renderizado 3D WebGL |
| **@react-three/fiber** | Bindings React para Three.js |
| **@react-three/drei** | Helpers 3D (controles, grilla) |
| **Tailwind CSS v4** | Estilos utilitarios |
| **mathjs** | Motor de cálculo matemático |

## 🚀 Cómo Iniciarlo

### Requisitos
- **Node.js** v18 o superior
- **npm** v8 o superior

### Pasos

```bash
# 1. Clona el repositorio
git clone https://github.com/TU_USUARIO/grafyapi.git

# 2. Entra a la carpeta
cd grafyapi

# 3. Instala dependencias
npm install

# 4. Inicia en modo desarrollo
npm run dev
```

Abre en el navegador: **`http://localhost:3000`** ✅

---

## 📦 Build de Producción

```bash
npm run build
npm start
```
